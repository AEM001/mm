# 行车规划问题详细报告

## 问题描述

某人开车从甲地（图中左下角1号路口）前往乙地（图中右上角100号路口）旅游，途中费用分餐饮住宿游览和汽油费两部分，餐饮住宿游览费用c=20t（元），其中时间t（小时）。汽车速度为v (公里/小时)时，每百公里耗油量V=0.0625v+1.875（升），假设路网所有加油站汽油单价均为7.76元/升。

图中纵横各有10条公路，任意相邻路口之间（称为路段）的距离均为50km，各路口编号以及各路段相应的速度上限（单位：公里/小时）标注在图中。标注为120的路段为高速公路，每公里收费0.5元。

问题一：在符合限速规定的条件下，求时间最短路线（路线一）的相应时间T1，以及花费最少路线(路线二)的费用C1。

## 一、模型分析

### 1.1 路网抽象

将10×10个路口看作图的100个节点（编号1…100），节点(i,j)对应编号(i-1)*10+j（i从下往上，j从左往右）。

- 横向边（左右移动）共有10×9条，限速由`limits_row[i][j]`给出。
- 纵向边（上下移动）共有9×10条，限速由`limits_col[i][j]`给出。

### 1.2 时间最短路径

在每条路段长度L=50 km，限速v_limit（km/h）条件下，过该路段所需时间：

t = L / v_limit

将每条边的"时间"作为权重，使用Dijkstra算法求1→100的最短路径，得到总时间T1。

### 1.3 费用最少路径

每条路段的总费用包括：

- 餐饮住宿游览：20元/时 × t
- 燃油费：油耗公式每百公里V=0.0625v+1.875（L），该速下50 km的油量=V×0.5，单价7.76元/L
- 高速通行费：限速=120 km/h时按0.5元/km×50 km=25元，否则0

因此在"以限速行驶"的假设下，费用权重：

c = 20×(L/v_limit) + (0.0625×v_limit+1.875)×(L/100)×7.76 + [v_limit=120]×25

同样用Dijkstra算法求最小费用路径，得到C1。

def build_graph():
    """

            if r > 1:
                v = node_index(r-1, c)
                speed_limit = limits_col[r-2][c-1]  # 注意索引转换
                adj[u].append((v, speed_limit, True))
            
            # 向右连接（如果不是最右列）
            if c < 10:
                v = node_index(r, c+1)
                speed_limit = limits_row[r-1][c-1]  # 注意索引转换
                adj[u].append((v, speed_limit, False))
            
            # 向左连接（如果不是最左列）
            if c > 1:
                v = node_index(r, c-1)
                speed_limit = limits_row[r-1][c-2]  # 注意索引转换
                adj[u].append((v, speed_limit, False))
    
    return adj

def index_to_rc(idx):
    """将节点索引（0-99）转换为行列坐标"""
    r = idx // 10 + 1  # 行（从下往上）
    c = idx % 10 + 1   # 列（从左往右）

### 2.2 权重函数定义

接下来，我们定义时间权重和费用权重函数：

```python
# === 三）权重函数 ===
def weight_time(limit, vertical):
    """计算时间权重（小时）"""
    return L / limit

def weight_cost(limit, vertical):
    """计算费用权重（元）"""
    # 时间成本
    t = L / limit
    c_time = time_cost_rate * t
    
    # 燃油成本
    fuel_consumption = (0.0625 * limit + 1.875) * (L / 100.0)  # 油耗（升）
    c_fuel = fuel_consumption * fuel_price
    
    # 高速通行费
    c_toll = toll_per_km * L if limit == 120 else 0.0
    
    return c_time + c_fuel + c_toll
```

### 2.3 Dijkstra算法实现

然后，我们实现Dijkstra算法来计算最短路径：

```python
# === 四）Dijkstra 算法 ===
def dijkstra(adj, start, end, weight_func):
    """
    使用Dijkstra算法计算最短路径
    
    参数:
        adj: 邻接表
        start: 起点索引
        end: 终点索引
        weight_func: 权重函数，接受(限速, 是否纵向)参数
        
    返回:
        (总权重, 路径)
    """
    n = len(adj)
    dist = [float('inf')] * n
    dist[start] = 0.0
    prev = [None] * n
    visited = [False] * n
    pq = [(0.0, start)]
    
    while pq:
        d, u = heapq.heappop(pq)
        
        if visited[u]:
            continue
        
        visited[u] = True
        
        if u == end:
            break
        
        for v, limit, vertical in adj[u]:
            if visited[v]:
                continue
                
            w = weight_func(limit, vertical)
            if dist[u] + w < dist[v]:
                dist[v] = dist[u] + w
                prev[v] = u
                heapq.heappush(pq, (dist[v], v))
    
    # 重建路径
    path = []
    u = end
    while u is not None:
        path.append(u)
        u = prev[u]
    path.reverse()
    
    return dist[end], path
```
